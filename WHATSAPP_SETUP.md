# WhatsApp-напоминания

KURSOR отправляет родителям напоминания через Green API. Поддерживаются автоматическая рассылка по расписанию, ручной запуск, тестовое сообщение, шаблоны и журнал результатов.

## Требования

- корпоративный аккаунт WhatsApp;
- активный Green API instance;
- заполненные телефоны родителей;
- расписание активных групп;
- `SETTINGS_ENCRYPTION_KEY` в production-конфигурации KURSOR.

## Подключение

1. Создайте instance в Green API.
2. Подключите корпоративный WhatsApp по QR-коду.
3. Скопируйте Instance ID и API token.
4. Откройте в KURSOR раздел «WhatsApp».
5. Введите Instance ID и token.
6. Сохраните настройки.
7. Отправьте тестовое сообщение.
8. Настройте время и включите автоматическую рассылку.

После сохранения API token больше не возвращается в браузер. В SQLite он хранится в формате AES-256-GCM. Если поле токена оставить пустым при следующем редактировании, сохранённый token не изменится.

Не меняйте `SETTINGS_ENCRYPTION_KEY` без процедуры ротации: старый ключ необходим для расшифровки уже сохранённого токена.

## Номера телефонов

Поддерживаются казахстанские номера в форматах:

- `+7 777 123 45 67`;
- `87771234567`;
- `77771234567`;
- `7771234567`.

Перед отправкой номер нормализуется. Некорректные номера отмечаются в журнале как пропущенные.

## Переменные шаблонов

| Переменная | Значение |
| --- | --- |
| `{{parentName}}` | Имя родителя |
| `{{studentName}}` | ФИО ученика |
| `{{groupName}}` | Название группы |
| `{{time}}` | Время занятия |
| `{{date}}` | Дата занятия |
| `{{weekday}}` | День недели |
| `{{address}}` | Адрес филиала |

## Правила рассылки

- преподаватель или ассистент может отправлять сообщения только по своим группам и только при наличии права `manage_whatsapp`;
- произвольное тестовое сообщение доступно только администратору;
- автоматическая рассылка не запускается параллельно сама с собой;
- успешное сообщение одному ученику и номеру не дублируется в течение суток;
- результаты и ошибки записываются в журнал WhatsApp.

## Диагностика

Если сообщение не отправлено:

1. проверьте статус instance;
2. убедитесь, что WhatsApp-телефон онлайн;
3. проверьте расписание и статус группы;
4. проверьте телефон в CRM-карточке;
5. откройте журнал отправок;
6. проверьте доступ сервера к Green API.
